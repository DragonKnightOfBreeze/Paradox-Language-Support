# Maintainer Documents

## 概述 / Overview

此目录用于存放项目的维护者文档。

不同于项目根目录下的普通文档以及 `docs` 目录下的参考文档，这些文档是面向项目维护者的。
其中可能涉及具体的架构细节和实现细节，以及需要进一步补充和整理的内容。

目录：
- [ai-notes](ai-notes) - AI 生成的笔记。包括领域相关、探索、报告、学习等类型的笔记。
- [notes](notes) - 维护者笔记。包括代码规范、架构决策、开发路线等。持续补充中。
- [references](references) - 针对特定主题的参考笔记。包括图标设计、配色方案等。未来可能补充更多主题。
- [changelogs](changelogs) - 已归档的历史更新日志。

备注：
- 此目录下的文档主要是以中文编写的。
- `ai-notes` 中的文档**基本上是按原样存档的**。项目的相关状态在归档后可能发生变化，例如其中涉及的代码位置、实现思路、架构决策等。
- `notes` 中的文档**必须完全由人工编写**。
- `references` 中的文档**必须完全由人工编写，或者经过人工审核和修订**。

---

This directory is used to store maintainer documentation for the project.

Different from the ordinary documents in the project root directory and the reference documents in the `docs` directory, these documents are for project maintainers.
They may involve specific architectural details and implementation details, as well as content that needs further addition and organization.

**Table of Contents:**
- **[ai-notes](ai-notes)** – AI-generated notes. Includes categories such as domain-related, exploration, reports, and learning notes.
- **[notes](notes)** – Maintainer notes. Includes code specifications, architectural decisions, development roadmap, etc. Continuously being supplemented.
- **[references](references)** – Reference notes on specific topics. Includes icon design, color schemes, etc. More topics may be added in the future.
- **[changelogs](changelogs)** – Archived historical update logs.

**Notes:**
- The documents in this directory are primarily written in Simplified Chinese. For non-Chinese readers, please get your translation engine or AI assistant ready. 
- The documents in `ai-notes` are **basically archived as is**. The relevant status of the project may have changed after archiving, such as the referenced code locations, implementation ideas, architectural decisions, etc.
- The documents in `notes` **must be entirely written by humans**.
- The documents in `references` **must be entirely written by humans, or have undergone human review and revision**.

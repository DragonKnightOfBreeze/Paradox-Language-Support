# 笔记：领域相关

## 名词翻译

预定义/动态生成

- scope → 作用域
- modifier → 修正
- trigger → 触发器
- effect → 效果

显式声明

- scripted variable → 封装变量
- definition → 定义
  - scripted trigger → 封装触发器
  - scripted effect → 封装效果
  - script value → 脚本值
  - define → 定值（区分于“定义” & 需要特指，尽管本质上更接近于“常量/参数/宏”）
  - game rule → 游戏规则
  - on action → 动作触发（最推荐的翻译）
  - event → 事件
  - event namespace → 事件命名空间
  - sprite -> 精灵
- localisation → 本地化
- directive → 指令
  - inline script → 内联脚本
  - definition injection → 定义注入

## 语言抽象

### 脚本文件

语法级别

- 封装变量（`ParadoxScriptScriptedVariable`）
- 成员（`ParadoxScriptMember`）
  - 属性（`ParadoxScriptProperty`）
  - 值（`ParadoxScriptValue`）
    - 布尔值（`ParadoxScriptBoolean`）
    - 数字（`ParadoxScriptNumber`）
    - 字符串（`ParadoxScriptString`）
    - 颜色（`ParadoxScriptColor`）
    - 封装变量引用（`ParadoxScriptScriptedVariable`）
    - 内联数学（`ParadoxScriptInlineMath`）
    - 块（`ParadoxScriptBlock`）
- 内联数学表达式（`ParadoxScriptInlineMathExpression`）
- 内联数学操作数（`ParadoxScriptInlineMathFactor`）
  - 内联数学数字（`ParadoxScriptInlineMathNumber`）
  - 内联数学封装变量引用（`ParadoxScriptInlineMathScriptedVariableReference`）
  - 内联数学参数（`ParadoxScriptInlineMathParameter`）
- 条件块（`ParadoxScriptParameterCondition`）
- 内联条件块（`ParadoxScriptInlineParameterCondition`）
- 条件表达式（`ParadoxScriptParameterConditionExpression`）
- 条件参数（`ParadoxScriptParameterConditionParameter`）
- 参数（`ParadoxScriptParameter`）

语义级别

- 封装变量（`scriptedVariable`）
- 定义（`definition`）
  - 定义的顶级键（`rootKey`）
  - 定义的类型键（`typeKey`）
  - 定义的成员（`definitionMember`）
- 指令（`directive`）
  - 内联指令（`inlineDirective`）
    - 实现：内联脚本用法（`inlineScriptUsage`） - Stellaris
  - 注入指令（`injectionDirective`）
    - 实现：定义注入（`definitionInjection`） - VIC3 / EU5
- 复杂表达式（`complexExpression`）
  - 实现：作用域字段表达式（`scopeFieldExpression`）
  - 实现：值字段表达式（`valueFieldExpression`）
  - ...

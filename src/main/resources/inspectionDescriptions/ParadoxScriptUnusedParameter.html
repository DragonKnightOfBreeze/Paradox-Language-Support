Reports unused parameters.

A parameter is considered as unused if it is set (by invocation expressions or script value expressions) or checked (by parameter condition expressions), but not used (in definition declarations).

Example:

# some_script_file.txt
some_definition = {
    some_prop = $PARAM$ # used
    [[UNUSED_PARAM] # checked but not used
        some_other_prop = $PARAM$
    ]
}

# some_other_script_file.txt
some_other_definition = {
    some_definition = {
        UNUSED_PARAM = value # set but not used
    }
    some_sv_prop = value:some_sv|UNUSED_PARAM|value| # set but not used
}

Based on plugin's built-in CWT Config.

Reports explicit scope calls in script files, which can be replaced with safe form.

Check on grammar level.

Applicable for game types that support safe assign operators (?= in CK3/VIC3/EU5, ? = in Stellaris).

Example:


# before
exists = owner
# comment
owner = { ... }

# [CK3/VIC3/EU5] after
# comment
owner ?= v

# [Stellaris] after
# comment
owner? = v

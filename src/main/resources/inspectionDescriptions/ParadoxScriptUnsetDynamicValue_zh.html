报告未设置的动态值。

如果动态值被使用（如 has-flag 表达式），但未被设置（如 set-flag 表达式），则视为未设置。

示例：


# some_script_file.txt
some_definition = {
    set_global_flag = flag
    some_prop = {
        limit = {
            has_global_flag = unset_flag # used but not set
        }
        # ……
    }
}

动态值是在匹配数据类型 Value、 ValueSet 或 DynamicValue 的脚本表达式中声明或使用的， 位于 @ 之前（如果存在）。

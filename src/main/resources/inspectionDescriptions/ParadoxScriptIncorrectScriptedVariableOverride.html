Reports (global) scripted variables that are overridden incorrectly according to their override rules.

Example:

The vanilla scripted variable is @var, declared in file common/scripted_variables/test.txt, and the overridden scripted variable is declared in file common/scripted_variables/zzz_test.txt.

This override is incorrect because the override rule is FIOS.

Based on plugin's built-in CWT Config.

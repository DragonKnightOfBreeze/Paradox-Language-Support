Reports unresolved scripted variable references in script files.

Scripted variable reference "SV" can be resolved to:

  • A local scripted variable of same name. (in the same file)
  • A global scripted variable of same name. (in a scripted variables script file.)

A scripted variables script file is a script file with extension ".txt" under directory common/scripted_variables or it's subdirectories.

Based on plugin's built-in CWT Config.

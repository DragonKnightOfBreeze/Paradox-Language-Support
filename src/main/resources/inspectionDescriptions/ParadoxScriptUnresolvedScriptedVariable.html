Reports that there are unresolved scripted variable references in a script file.

a scripted variable reference can be resolved to:

  • a local scripted variable of same name. (in the same file outside of definition declaration)
  • a global scripted variable of same name. (in a scripted variables script file.)

A scripted variables script file is a script file with extension ".txt" under directory gfx/interface or it's sub directories relative to the game or mod root directory.

Based on plugin's built-in CWT Config currently.

Reports incorrect event namespace declarations in an event script file. Event namespace should be in correct format.

In detail, an event namespace should only contain digits and underlines.

In an event script file, the event namespace is declared as the root property of the file before event definitions, with name"namespace".

An event script file is a script file with extension ".txt" under directory events or it's subdirectories relative to the game or mod root directory.

Based on plugin's built-in CWT Config currently.

NOTE: Support multiple event namespaces in the same event script file.

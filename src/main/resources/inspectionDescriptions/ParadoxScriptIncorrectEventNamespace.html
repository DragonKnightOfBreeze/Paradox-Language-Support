Checks that the event id of each event definition declared in an event script file matches the event namespace.

In detail, an event namespace should only contains digits and underlines.

In an event script file, the event namespace is declared as the root property of the file before event definitions, with name"namespace".

An event script file is a script file under directory events relative to the game or mod root directory.

NOTE: Support multiple event namespaces in the same event script file.

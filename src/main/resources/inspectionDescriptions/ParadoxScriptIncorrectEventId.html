Reports incorrect event id declarations in related event declarations.

Event id should be in correct format.

In detail, the format of an event id should be "{namespace}.{no}", while "{namespace}" is the event namespace (ignore case), and "{no}" is a non-negative integer (ignore leading zero).

In an event script file, the event id is declared as a property of an event definition (named "id"), the event namespace is declared as the root property of the file (named "namespace") before event definitions.

An event script file is a script file with extension ".txt" under directory events.

Checks that the event id of each event definition declared in an event script file matches the event namespace.

In detail, the format of an event id should be "{namespace}.{no}", while "{namespace}" is the event namespace (ignore case), and "{no}" is a non-negative integer (ignore leading zero).

In an event script file, the event id is declared as a property of an event definition, with name "id", the event namespace is declared as the root property of the file before event definitions, with name"namespace".

An event script file is a script file under directory events relative to the game or mod root directory.

NOTE: Support multiple event namespaces in the same event script file.

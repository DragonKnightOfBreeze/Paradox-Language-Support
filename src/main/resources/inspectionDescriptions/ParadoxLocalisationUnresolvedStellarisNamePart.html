Reports unresolved stellaris name part references in localisation files.

Stellaris name part reference "part" can be resolved to:

  • A stellaris name part of same name.

A stellaris name part is a script expression matching CWT expression "value_set[xxx_name_parts]", and it's also some name parts list's definition name (aka key).

In a localisation file under directory localisation / localization or it's subdirectories, the stellaris name part reference syntax is supported in the localisation text only if the localisation's name starts with "format.".

If the localisation is referenced in script files by CWT expression "stellaris_name_format[xxx]", the stellaris name part references in its localisation text can be resolved to script expressions matching CWT expression "value_set[xxx]".

In an event script file, the event id is declared as a property of an event definition (named "id"), the event namespace is declared as the root property of the file (named "namespace") before event definitions.

Based on plugin's built-in CWT Config.

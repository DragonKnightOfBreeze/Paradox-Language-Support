Reports chained scope calls in script files, which can be replaced with nested form.

Check on grammar level and semantic level.

Example (Use __ to represent cursor position):


# before
__root.owner = { a = 1 }

# after
__root = {
    owner = { a = 1 }
}

# before
root.__owner = { a = 1 }

# after
root = {
    __owner = { a = 1 }
}

# before
root.__owner.event_target:x

# after
root = {
    __owner.event_target:x
}

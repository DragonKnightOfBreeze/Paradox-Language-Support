Reports chained scope calls in script files, which can be replaced with nested form.

Check on grammar level and semantic level.

Example (Use <caret> to represent cursor position):


# before
<caret>root.owner = { a = 1 }

# after
<caret>root = {
    owner = { a = 1 }
}

# before
root.<caret>owner = { a = 1 }

# after
root = {
    <caret>owner = { a = 1 }
}

# before
root.<caret>owner.event_target:x

# after
root = {
    <caret>owner.event_target:x
}

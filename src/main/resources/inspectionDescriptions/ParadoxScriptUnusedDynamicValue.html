Reports unused dynamic values.

A dynamic value is considered as unused if it is set (by, such as, set-flag expressions) but not used (by, such as, has-flag expressions).

Example:


# some_script_file.txt
some_definition = {
    set_global_flag = unused_flag # set but not used
    some_prop = {
        limit = {
            has_global_flag = flag
        }
        # ...
    }
}

A dynamic value is declared or used in a script expression matching data type Value, ValueSet, or DynamicValue, where before @ if it exists.

Reports unresolved command field references in localisation files.

Localisation command field reference "FIELD" can be following situations currently:

  • A localisation command named "FIELD" (ignore case).
  • A script loc definition named "FIELD" (ignore case).
  • A variable named "FIELD" (ignore case).

Localisation commands are predefined by CWT Config (in localisation.cwt#localisation_commands)

A script loc definition is a definition with type "script_loc".

A variable is a script expression matching CWT expression "value[variable]" or "value_set[variable]".

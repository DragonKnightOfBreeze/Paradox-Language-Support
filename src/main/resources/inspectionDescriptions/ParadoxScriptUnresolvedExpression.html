Reports unresolved expressions in definition declarations.

A expression (key / value) in a definition declaration will be considered as unresolved in following situations:

  • It's a key expression, but does not match any CWT expression.
  • It's a key expression, but value expression of it does not match any CWT expression.
  • It's a value expression, but does not match any CWT expression.

Based on plugin's built-in CWT Config.

Reports unresolved expressions in script files.

An expression (key / value) in a definition declaration will be considered as unresolved in following situations:

  • It's a key expression, but does not match any config expression.
  • It's a key expression, but value expression of it does not match any config expression.
  • It's a value expression, but does not match any config expression.

NOTE: This code inspection is based on config files, so if it produces false positives, it usually indicates that the config files for current game type is incomplete. Consider disabling or suppressing this code inspection if needed, or contributing configs.

NOT IMPLEMENTED

Reports unexpected script expressions and statements in a script file.

  • Unexpected inline math. (Expect in a definition declaration)
  • Unexpected parameter. (Expect in a definition declaration)
  • Unexpected parameter condition statement. (Expect in a definition declaration)
  • Unexpected inline math start token. (Expect"@\[" in a scripted_effect / scripted_trigger definition declaration, otherwise"@[")
  • Multiple inline maths in single scripted_effect/ scripted_trigger definition declaration.

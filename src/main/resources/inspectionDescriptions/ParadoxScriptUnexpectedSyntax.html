Reports that there are unexpected script expressions and statements in a script file.
  • Unexpected inline math. (Expect in a definition declaration)
  • Unexpected parameter. (Expect in a definition declaration)
  • Unexpected parameter condition statement. (Expect in a definition declaration)
  • Unexpected inline math start token. (Expect"@\[" in a scripted effect/trigger definition declaration, otherwise"@[" )
  • Multiple inline maths in single definition declaration.

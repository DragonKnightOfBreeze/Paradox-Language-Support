Reports unresolved command scope references in localisation files.

Localisation command scope reference "SCOPE" can be following situations currently:

  • A system link named "SCOPE" (ignore case).
  • A localisation scope named "SCOPE" (ignore case).
  • A event target named "SCOPE" (ignore case).
  • A global event target named "SCOPE" (ignore case).

System links are predefined by CWT Config (in system_links.pls.cwt).

Localisation scopes are predefined by CWT Config (in localisation.cwt#localisation_links)

An event target is a script expression matching CWT expression "value[event_target]" or "value_set[event_target]".

A global event target is a script expression matching CWT expression "value[global_event_target]" or "value_set[global_event_target]".

Reports inline scripts with conflicting usages.

Usages of an inline script will be considered as conflicting if they have conflicting config contexts. (e.g. alias_name[trigger] VS alias_name[effect])

An inline script file is a script file with extension .txt under directory common/inline_scripts. Then it can be used in other script files (almost everywhere).

Example:


# some_inline_script.txt (under directory common/inline_scripts/dir)
# ...

# some_other_script_file.txt
some_other_definition = {
    inline_script = "dir/some_inline_script"
}

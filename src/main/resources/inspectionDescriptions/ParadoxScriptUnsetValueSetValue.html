Reports unset value set values.

A value set value is considered as unset if it is used (by, such as, has-flag expression) but not set (by, such as, set-flag expression).

Example:

# some_script_file.txt
some_definition = {
    set_global_flag = flag
    some_prop = {
        limit = {
            has_global_flag = unset_flag # used but not set 
        }
        # ...
    }
}

A value set value is a script key/value expression with CWT data type value/value_set.

Based on plugin's built-in CWT Config.

Reports unset value set values.

A value set value is considered as unset if it is used (by, such as, has-flag expressions) but not set (by, such as, set-flag expressions).

Example:

# some_script_file.txt
some_definition = {
    set_global_flag = flag
    some_prop = {
        limit = {
            has_global_flag = unset_flag # used but not set 
        }
        # ...
    }
}

A value set value is declared or used in a script expression matching CWT expression data type "value" or "value_set", before "@" if it exists.

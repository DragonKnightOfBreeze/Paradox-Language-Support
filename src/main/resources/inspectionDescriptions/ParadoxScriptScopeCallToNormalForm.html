Reports scope calls with safe form that can be replaced with explicit form.

Check on grammar level.

Supported for all game types with any safe call operator (?= or ? =).

Description:

  • For explicit call form, exists = x must be before x = { ... }, and there can only be blank space or comments between them.

Example - explicit form:


exists = owner
owner = { ... }

Example - safe form:

[CK3/VIC3/EU5]


owner ?= { ... }

Stellaris:


owner? = { ... }

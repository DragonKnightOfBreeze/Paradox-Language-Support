Report script files that cannot match any configs.

The currently checked config types include type configs and complex enum configs.

Ignore injected files and temporary files. Additionally, files located directly under the game or entry directory are also ignored.

NOTE: This code inspection is based on config files, so if a false positive occurs, it indicates that the current file is special or ignorable, or that the config files for current game type is incomplete. Consider disabling this code inspection or excluding file paths as needed, or contributing configs.

Report script files that cannot match any configs.

The currently checked config types include type configs and complex enum configs.

Description:

  • Ignore injected files and temporary files.
  • Ignore files located directly in the game or portal directory.
  • Ignore files used to declare defines, global scripted variables, or inline scripts. They are located in the common/defines common/scripted_variables and common/inline_scripts directories respectively.

NOTE: This code inspection is based on config files, so if it produces false positives, it indicates that the current file is special or ignorable, or that the config files for current game type is incomplete. Consider disabling or suppressing this code inspection as needed, or excluding relevant file paths, or contributing configs.

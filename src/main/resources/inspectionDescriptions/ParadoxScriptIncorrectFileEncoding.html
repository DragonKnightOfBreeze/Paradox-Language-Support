Reports script files with incorrect file encoding.

In detail, it should be UTF-8.

Specially:

  • For Stellaris, it should be UTF-8 BOM for script files under directory common/name_lists, otherwise UTF-8 NO BOM.
  • For other game types, it may use or not use BOM, depends on the game type and the file path, this inspection does not verify these situations atm.

Ignore injected files and temporary files.

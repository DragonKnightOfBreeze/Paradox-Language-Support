Reports that there are unresolved icon references in a localisation file.

Name of an localisation icon can be blow situations currently:

  • "icon_name" if there exists an sprite definition named "GFX_text_icon_name".
  • "icon_name" if there exists an sprite definition named "GFX_icon_name".
  • "dds_name" if there exists a DDS file named "dds_name.dds" under directory gfx/interface/icons or it's sub directories relative to the game or mod root directory.
  • Generated localisation icon names. NOT SUPPORTED YET and will report as an error.

An sprite script file is a script file with extension name ".gfx" under directory gfx/interface or it's sub directories relative to the game or mod root directory.

Reports that there are unresolved icon references in a localisation file.

Name of an localisation icon can be blow situations currently:

  • "icon_name" if there exists a sprite definition named "GFX_text_icon_name".
  • "icon_name" if there exists a sprite definition named "GFX_icon_name".
  • "dds_name" if there exists a DDS file named "dds_name.dds" under directory gfx/interface/icons or it's sub directories relative to the game or mod root directory.
  • Generated localisation icon names. Code navigation and completion is NOT SUPPORTED YET.

A sprite script file is a script file with extension ".gfx" under directory common/scripted_variables or it's sub directories relative to the game or mod root directory.

Based on plugin's built-in CWT Config currently.

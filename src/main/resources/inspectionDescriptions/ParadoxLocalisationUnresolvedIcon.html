Reports unresolved icon references in localisation files.

Localisation icon reference "icon_name" can be resolved to following situations currently:

  • A text sprite definition (type of which is "sprite") named "GFX_text_icon_name".
  • A sprite definition (type of which is "sprite") named "GFX_icon_name".
  • A DDS file named "icon_name.dds" under directory gfx/interface/icons.

A sprite definition is a definition with type "sprite" or "sprite_type" (base on game type) by CWT Config.

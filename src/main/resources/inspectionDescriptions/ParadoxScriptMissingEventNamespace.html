Reports event script files which miss event namespace declarations.

Description:

  • This code inspection is heuristic and may contain false positives.
  • When this code inspection fails, it does not necessarily mean that an exception will be thrown at the game engine level.
  • Actually, an event script file can declare no or multiple event namespaces, and the event id does not need to strictly match the previously declared event namespace in the same file.

In an event script file, the event namespace is declared as the root property of the file before event definitions, while the property's name is namespace.

An event script file is a script file with extension .txt under directory events.

Reports a localisation file that there are multiple locale declarations in it.

For a localisation file, it's name should contains a locale id (e.g. "l_english"), it's content should only contain the matched locale declaration (e.g. "l_english" in file events_l_english.yml).

Localisation locales are predefined by Internal Config (in localisation_config.pls.cwt#locales).

Based on plugin's built-in Internal Config.

Reports mismatched event id declarations in related event declarations.

Event id should match the event namespace declared before it in event script files.

In an event script file, the event id is declared as a property of an event definition (named "id"), the event namespace is declared as the root property of the file (named "namespace") before event definitions.

An event script file is a script file with extension ".txt" under directory events or it's subdirectories.

NOTE: In real, event id and event namespace can be matched crossing event script files in game loading order.

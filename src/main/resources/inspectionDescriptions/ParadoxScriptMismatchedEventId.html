Report event ids in event script files that do not match the event namespace, which are located in event declarations.

Description:

  • This code inspection is heuristic and may contain false positives.
  • When this code inspection fails, it does not necessarily mean that an exception will be thrown at the game engine level.
  • Actually, An event script file can declare no or multiple event namespaces, and the event id does not need to match the previously declared event namespace in the same file.

In an event script file, the event id is declared as the key or a property (named id) value of an event definition, the event namespace is declared as the root property of the file (named namespace) before event definitions.

An event script file is a script file with extension .txt under directory events.

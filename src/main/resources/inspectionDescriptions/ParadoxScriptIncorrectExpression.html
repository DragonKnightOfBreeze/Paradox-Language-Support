Reports incorrect expressions in definition declarations.

An expression (key / value) will be considered as incorrect in following situations:

  • It's a number matched by a CWT expression, but its value is outside the number range declared by this CWT expression.
  • It's a color field matched by a CWT expression, but its color type does not match the color type declared by this CWT expression.
  • It's a scope field matched by a CWT expression, but its value does not match the scopes declared by this CWT expression.
  • Other specific situations.

Based on plugin's built-in CWT Config.

Reports unused value set values.

A value set value is considered as unused if it is set (by, such as, set-flag expressions) but not used (by, such as, has-flag expressions).

Example:

# some_script_file.txt
some_definition = {
    set_global_flag = unused_flag # set but not used
    some_prop = {
        limit = {
            has_global_flag = flag 
        }
        # ...
    }
}

A value set value is declared or used in a script expression matching CWT expression data type "value" or "value_set", before "@" if it exists.

Based on plugin's built-in CWT Config.

Reports unused value set values.

A value set value is considered as unused if it is set (by, such as, set-flag expression) but not used (by, such as, has-flag expression).

Example:

# some_script_file.txt
some_definition = {
    set_global_flag = unused_flag # set but not used
    some_prop = {
        limit = {
            has_global_flag = flag 
        }
        # ...
    }
}

A value set value is a script key/value expression with CWT data type value/value_set.

Based on plugin's built-in CWT Config currently.

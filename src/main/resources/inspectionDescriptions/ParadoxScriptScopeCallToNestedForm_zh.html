报告脚本文件中的链式的作用域调用，可以被替换为嵌套形式。

检测于文法级别和语义级别。

示例（使用 __ 表示光标位置）：


# before
__root.owner = { a = 1 }

# after
__root = {
    owner = { a = 1 }
}

# before
root.__owner = { a = 1 }

# after
root = {
    __owner = { a = 1 }
}

# before
root.__owner.event_target:x

# after
root = {
    __owner.event_target:x
}

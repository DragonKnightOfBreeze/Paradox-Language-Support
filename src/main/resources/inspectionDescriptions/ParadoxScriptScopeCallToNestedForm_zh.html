报告脚本文件中的链式的作用域调用，可以被替换为嵌套形式。

检测于文法级别和语义级别。

示例（<caret> 表示光标位置）：


# before
<caret>root.owner = { a = 1 }

# after
<caret>root = {
    owner = { a = 1 }
}

# before
root.<caret>owner = { a = 1 }

# after
root = {
    <caret>owner = { a = 1 }
}

# before
root.<caret>owner.event_target:x

# after
root = {
    <caret>owner.event_target:x
}

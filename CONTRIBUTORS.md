## CWT Config Maintenance

The plugin's config-driven features (code completion, code inspection, quick documentation, etc.) depend on CWT config files being up to date with each game version.
The table below shows the current maintenance status for each supported game.

## Config Repositories

Each game has two config repositories — **Built-in** (used by PLS, may contain PLS-specific extensions) and **Vanilla** (the original CWTools repository, or the upstream source).

| Game | Built-in (PLS) | Vanilla (CWTools / Upstream) |
|------|----------------|------------------------------|
| Stellaris | [cwtools-stellaris-config](https://github.com/DragonKnightOfBreeze/cwtools-stellaris-config) | [cwtools/cwtools-stellaris-config](https://github.com/cwtools/cwtools-stellaris-config) |
| Victoria 3 | [cwtools-vic3-config](https://github.com/DragonKnightOfBreeze/cwtools-vic3-config) | [cwtools/cwtools-vic3-config](https://github.com/cwtools/cwtools-vic3-config) |
| Europa Universalis V | [cwtools-eu5-config](https://github.com/DragonKnightOfBreeze/cwtools-eu5-config) | [kaiser-chris/cwtools-eu5-config](https://github.com/kaiser-chris/cwtools-eu5-config) |
| Crusader Kings III | [cwtools-ck3-config](https://github.com/DragonKnightOfBreeze/cwtools-ck3-config) | [cwtools/cwtools-ck3-config](https://github.com/cwtools/cwtools-ck3-config) |
| Hearts of Iron IV | [cwtools-hoi4-config](https://github.com/DragonKnightOfBreeze/cwtools-hoi4-config) | [cwtools/cwtools-hoi4-config](https://github.com/cwtools/cwtools-hoi4-config) |
| Europa Universalis IV | [cwtools-eu4-config](https://github.com/DragonKnightOfBreeze/cwtools-eu4-config) | [cwtools/cwtools-eu4-config](https://github.com/cwtools/cwtools-eu4-config) |
| Imperator Rome | [cwtools-ir-config](https://github.com/DragonKnightOfBreeze/cwtools-ir-config) | [cwtools/cwtools-ir-config](https://github.com/cwtools/cwtools-ir-config) |
| Crusader Kings II | [cwtools-ck2-config](https://github.com/DragonKnightOfBreeze/cwtools-ck2-config) | [cwtools/cwtools-ck2-config](https://github.com/cwtools/cwtools-ck2-config) |
| Victoria 2 | [cwtools-vic2-config](https://github.com/DragonKnightOfBreeze/cwtools-vic2-config) | [cwtools/cwtools-vic2-config](https://github.com/cwtools/cwtools-vic2-config) |

Apart from Stellaris (whose PLS configs have diverged significantly with many additions), configs for other games are largely kept in sync between CWTools and PLS.
PLS-specific extensions in the built-in configs should not cause errors on the CWTools side.
